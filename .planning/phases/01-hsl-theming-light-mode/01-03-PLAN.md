**Part A: Update ThemeContext.jsx — add guard to useTheme**

Replace the current `useTheme` export (line 29):
```js
export const useTheme = () => useContext(ThemeContext)
```

```js
export function useTheme() {
  const ctx = useContext(ThemeContext)
```
