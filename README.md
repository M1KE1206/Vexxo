# Vexxo

Vexxo is opgebouwd met React via Next.js 14 en gebruikt Tailwind CSS voor styling.

## Stack

- React 18
- Next.js 14
- Tailwind CSS 3
- TypeScript

## Scripts

- `npm run dev` start de development server
- `npm run build` maakt een production build
- `npm run start` start de production server
- `npm run lint` controleert de code met Next.js ESLint
